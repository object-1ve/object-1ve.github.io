存储容量为8K×8的ROM（只读存储器）中的”8K”和”8”分别代表了两个不同的属性：

  1. 8K：这里的”K”代表千（kilo），所以8K表示8000。这意味着ROM有8000个存储单元（或地址）可以访问。每个存储单元可以存储一个数据项。
  2. 8：这个数字表示每个存储单元可以存储的数据位数。在这个例子中，每个存储单元可以存储8位（bit）的数据，即一个字节（byte）。因为1 byte = 8 bits。

综合来看，8K×8的ROM意味着这个存储器有8000个存储位置，每个位置可以存储8位数据，总共可以存储的数据量是8000字节。这种存储器通常用于存储固件或系统软件，这些软件在设备运行时不需要频繁更改。

1. 布尔代数、逻辑代数

布尔代数是用于逻辑运算的一种代数结构，主要包含以下元素和运算：

  • 0 和 1: 代表布尔值，0 通常表示假，1 通常表示真。
  • 正逻辑: 1 表示真，0 表示假。
  • 负逻辑: 1 表示假，0 表示真。

2. 进制和码制转换

常见进制和码制包括二进制、八进制、十六进制、十进制以及 8421BCD 码。各进制之间的转换关系如下：

  • 二进制（Binary）: 基数为 2，用 0 和 1 表示。
  • 八进制（Octal）: 基数为 8，用 0-7 表示。每个八进制数对应 3 位二进制数。
  • 十六进制（Hexadecimal）: 基数为 16，用 0-9 和 A-F 表示。每个十六进制数对应 4 位二进制数。
  • 十进制（Decimal）: 基数为 10，用 0-9 表示。
  • 8421BCD 码: 每个十进制数用 4 位二进制数表示。

转换示例：

  • 二进制到十六进制: 二进制数 1101 1010 转换为十六进制数为 DA。
  • 十六进制到二进制: 十六进制数 B 转换为二进制数为 1011。
  • 8421BCD 码: 十进制数 19 转换为 8421BCD 码为0001 1001。

3. 基本逻辑关系

基本逻辑运算包括与（AND）、或（OR）、非（NOT），扩展逻辑门包括与非门（NAND）、或非门（NOR）、异或门（XOR）、同或门（XNOR）。

  • 与（AND）: 两个输入都为 1 时输出为 1，否则输出为 0。
  • 或（OR）: 只要有一个输入为 1，输出为 1。
  • 非（NOT）: 输入为 0 输出为 1，输入为 1 输出为 0。
  • 与非门（NAND）: 与运算后取反。
  • 或非门（NOR）: 或运算后取反。
  • 异或门（XOR）: 输入相同时输出为 0，输入不同时输出为 1。
  • 同或门（XNOR）: 输入相同时输出为 1，输入不同时输出为 0。

4. 真值表、逻辑表达式、最小项和标准与或式

  • 真值表: 显示逻辑运算中所有输入组合的输出结果。
  • 逻辑表达式: 使用逻辑运算符（如 AND、OR、NOT）表示逻辑关系。
  • 最小项: 对于 n 个变量，每一个变量的组合（如 00、01、10、11）称为最小项。
  • 标准与或式: 用最小项组合成的逻辑表达式。
  • 最简式: 通过化简逻辑表达式得到的简洁形式。

5. 公式定理规则

  • 代入规则: 根据已有的逻辑表达式进行代入计算。
  • 反演规则: 使用德摩根定律进行反演操作。

公式化简法

公式化简通常使用以下方法：

  1. 代入规则: 替换表达式中的变量。
  2. 结合律和分配律: 进行组合和分配以简化表达式。
  3. 反演规则: 使用德摩根定律等进行反演简化。
  4. Karnaugh 图法: 通过 Karnaugh 图（卡诺图）直观地进行化简。

6. 卡诺图

卡诺图（Karnaugh Map，K-Map）是一种用于化简布尔代数表达式的工具。通过图形化的方法，可以轻松识别和消除冗余项。

  • 逻辑相邻项: 在卡诺图中，逻辑相邻项指的是仅有一个变量不同的相邻格子。例如，对于两个变量 A 和 B，它们的四个组合（00、01、11、10）在卡诺图中排列成一个 2x2 的矩阵，任意相邻格子代表逻辑相邻项。
  • 几何相邻: 直接相邻或通过边界相邻的格子。例如，边界相邻的格子 00 和 10 可以视为相邻。

卡诺图化简法

  • 圈越大越好: 尽可能圈住多个“1”来消除更多的变量。
  • 圈的个数越少越好: 用最少的圈覆盖所有的“1”。
  • 每个圈内有不同的“1”: 每个圈应该尽可能覆盖不同的“1”。

7. 真值表、逻辑表达式、电路图、波形图

  • 真值表: 显示输入变量的所有可能组合及其对应的输出。
  • 逻辑表达式: 用布尔代数表示输出与输入之间的关系。
  • 电路图: 用电子元件符号表示逻辑电路的连接方式。
  • 波形图: 显示输入和输出随时间变化的波形。

8. 二极管导通条件

  • 正向导通: 二极管正向偏置时（阳极电压高于阴极电压），导通电压约为 0.7V（硅二极管）。

9. 组合逻辑电路分析

步骤如下：

  1. 电路图 -> 表达式: 从电路图中找出各个逻辑门的关系，写出逻辑表达式。
  2. 表达式 -> 真值表: 根据逻辑表达式，列出真值表。
  3. 真值表 -> 描述功能: 根据真值表，描述电路的功能。

10. 组合逻辑电路设计

步骤如下：

  1. 功能 -> 逻辑抽象: 根据功能需求，定义输入和输出。
  2. 逻辑抽象 -> 真值表: 列出所有输入组合及对应输出。
  3. 真值表 -> 表达式: 从真值表中得出逻辑表达式。
  4. 表达式 -> 根据要求转换表达式: 化简逻辑表达式或转换为特定逻辑门形式。
  5. 表达式 -> 电路图: 画出对应的逻辑电路图。

11. 典型的组合逻辑电路

  • 比较器: 比较两个二进制数的大小，输出较大或相等的信号。
  • 加法器: 实现二进制数的加法运算，常见的有半加器和全加器。
  • 译码器: 将二进制码转换为唯一的输出信号。
  • 数据分配器: 将单一输入信号分配到多个输出。
  • 数据选择器: 从多个输入信号中选择一个输出。
  • 译码显示器: 将 8421BCD 码转换成七段数码管显示。

12. 用中规模集成电路设计任意逻辑电路

  • 使用译码器或数据选择器设计复杂的逻辑电路。
  • 可以通过级联多个译码器或数据选择器实现更大的逻辑范围。

13. 触发器

触发器是具有记忆功能的基本单元，用于构建时序逻辑电路。常见触发器有：

  • RS 触发器: 基于与非门（NAND）或或非门（NOR）的简单触发器。输入为 R 和 S，具有置位和复位功能。
  • D 触发器: 数据触发器，输入为 D 和时钟信号，D 输入直接决定输出状态。
  • JK 触发器: 改进型 RS 触发器，输入为 J 和 K，解决了 RS 触发器的不确定状态。
  • T 触发器: 翻转触发器，输入为 T，T=1 时翻转输出状态，T=0 时保持当前状态。

触发器的特性方程及转换

  • RS 触发器: ( Q_{next} = \overline{R} \cdot (S + \overline{Q}) )
  • D 触发器: ( Q_{next} = D )
  • JK 触发器: ( Q_{next} = J \cdot \overline{Q} + \overline{K} \cdot Q )
  • T 触发器: ( Q_{next} = T \oplus Q )

触发器的功能及波形图

每种触发器根据输入信号和时钟信号的变化产生对应的输出波形，显示触发器的状态变化过程。

14. 典型时序逻辑电路

  • 计数器: 计数脉冲信号的个数。
  • 顺序脉冲发生器: 按预定顺序产生脉冲信号。
  • 移位寄存器: 按时钟信号将数据依次移位存储。
  • 环形计数器: 移位寄存器的一种，最后一个输出连接到第一个输入。
  • 扭环形计数器: 改进的环形计数器，增加了反向反馈。

这些时序逻辑电路在数字系统中应用广泛，用于实现各种控制和数据处理功能。
